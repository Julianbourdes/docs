## Planning Poker

Organisez des sessions d'estimation efficaces avec votre équipe.

### Votez Ensemble, Décidez Plus Vite

Démarrez une session et partagez votre code unique. Votre équipe rejoint instantanément et vote en même temps. Voyez les résultats dès que tout le monde a voté.

### Des Decks d'Estimation qui Fonctionnent pour Vous

Choisissez l'échelle d'estimation qui convient à votre équipe :
- **Fibonacci** - Estimation agile classique (1, 2, 3, 5, 8, 13...)
- **Tailles T-Shirt** - Dimensionnement simple (XS, S, M, L, XL)
- **Puissances de 2** - Estimations techniques (1, 2, 4, 8, 16...)
- **Decks Personnalisés** - Créez vos propres valeurs et sauvegardez pour réutiliser

### Connectez Votre Espace Jira

Importez des stories directement depuis Jira :
- Parcourez vos projets et choisissez les stories dont vous avez besoin
- Filtrez par type d'issue, statut ou assigné
- Les estimations se synchronisent automatiquement dans Jira quand vous avez terminé

### Organisez Vos Sessions

- **Codes de Participation Uniques** - Partagez un simple code de 6 caractères pour inviter n'importe qui
- **Archive des Sessions** - Accédez aux sessions passées à tout moment
- **Options de Vote** - Choisissez le vote anonyme, la révélation auto ou affichez les cartes pendant le vote
- **Minuteurs de Vote** - Définissez des limites de temps pour que les sessions avancent

### Gardez Tout le Monde Engagé

- **Gestion des Rôles** - Attribuez des hôtes, participants et spectateurs
- **Mises à Jour en Direct** - Voyez qui a voté et qui est encore en train de décider
- **Mode Anonyme** - Masquez les votes individuels jusqu'à ce que tout le monde soit prêt
- **Vue Spectateur** - Laissez les parties prenantes observer sans voter

## Travaillez avec Votre Équipe

### Gérez Plusieurs Organisations

Passez d'une entreprise ou d'un département à l'autre depuis un seul compte. Chaque organisation garde ses propres équipes, sessions et paramètres.

### Invitez Votre Équipe

Envoyez des invitations par email à vos collègues. Ils choisiront comment se connecter quand ils rejoindront.

### Attribuez les Bons Rôles

- **Admins d'Organisation** gèrent les paramètres et les membres de l'équipe
- **Chefs d'Équipe** créent et organisent des sessions d'estimation
- **Membres** participent et collaborent

## Connectez-vous en Toute Sécurité

### Choisissez Votre Mode de Connexion Préféré

Choisissez ce qui vous convient :
- **Google** - Connectez-vous avec votre compte Google
- **Microsoft** - Utilisez vos identifiants Microsoft 365
- **GitHub** - Connectez-vous avec GitHub

### Liez Plusieurs Comptes

Connectez Google, Microsoft et GitHub au même compte Teamova. Passez de l'un à l'autre à tout moment.

### Définissez un Mot de Passe de Secours

Même si vous vous êtes inscrit avec Google, Microsoft ou GitHub, vous pouvez ajouter un mot de passe comme méthode de connexion alternative.

## Options de Langue

Utilisez Teamova dans votre langue :
- **English** - Interface complète en anglais
- **Français** - Version française complète

Changez de langue dans vos paramètres de compte.
